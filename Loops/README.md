# 🐍 Loops

## 📖 About
This folder contains Python programs related to loops. Loops are used to execute a block of code repeatedly, making programs more efficient and reducing repetitive code.

## 📚 Topics Covered
- for Loop
- while Loop
- Nested Loops
- break Statement
- continue Statement
- pass Statement
- Range Function
- Loop Control Statements

## 🎯 Goal
To understand how loops work in Python and use them to solve repetitive tasks efficiently.

After completing this folder, I will be able to use loops effectively to automate repetitive tasks, control program flow, and solve programming problems efficiently.
