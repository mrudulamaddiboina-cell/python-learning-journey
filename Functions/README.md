# 🐍 Functions

## 📖 About
This folder contains Python programs related to functions. Functions help organize code into reusable blocks, making programs easier to read, maintain, and debug.

## 📚 Topics Covered
- Function Definition
- Function Calling
- Parameters and Arguments
- Return Statements
- Default Arguments
- Keyword Arguments
- Variable-Length Arguments (*args, **kwargs)
- Lambda Functions
- Recursive Functions

## 🎯 Goal
To understand how functions work in Python and learn to write reusable, modular, and efficient code.

## 📂 Programs
- function_definition.py
- function_calling.py
- parameters_arguments.py
- return_statement.py
- default_arguments.py
- keyword_arguments.py
- lambda_functions.py
- recursive_functions.py

## ✍️ Learning Outcome
After completing this folder, I will be able to create, call, and reuse functions effectively while writing clean and organized Python programs.
