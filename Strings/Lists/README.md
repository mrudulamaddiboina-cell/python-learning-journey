# 🐍 Lists

## 📖 About
This folder contains Python programs related to lists. Lists are one of the most commonly used data structures in Python for storing and manipulating collections of data.

## 📚 Topics Covered
- Creating Lists
- Accessing List Elements
- List Slicing
- Updating Lists
- Adding Elements (append, extend, insert)
- Removing Elements (remove, pop, clear)
- List Methods
- List Traversal
- Nested Lists
- List Comprehension

## 🎯 Goal
To understand how to create, access, modify, and manipulate lists efficiently in Python.

## 📂 Programs
- create_list.py
- access_elements.py
- list_slicing.py
- update_list.py
- add_elements.py
- remove_elements.py
- list_methods.py
- nested_lists.py
- list_comprehension.py

## ✍️ Learning Outcome
After completing this folder, I will be able to work confidently with Python lists, perform common list operations, and solve problems using lists.
