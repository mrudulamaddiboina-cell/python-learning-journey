# 🐍 Tuples

## 📖 About
This folder contains Python programs related to tuples. Tuples are ordered and immutable collections used to store multiple items efficiently.

## 📚 Topics Covered
- Creating Tuples
- Accessing Tuple Elements
- Tuple Slicing
- Tuple Packing and Unpacking
- Tuple Methods (count, index)
- Nested Tuples
- Iterating Through Tuples
- Tuple Operations

## 🎯 Goal
To understand how tuples work in Python and learn when to use them instead of lists.

## 📂 Programs
- create_tuple.py
- access_elements.py
- tuple_slicing.py
- packing_unpacking.py
- tuple_methods.py
- nested_tuples.py
- iterate_tuples.py
- tuple_operations.py

## ✍️ Learning Outcome
After completing this folder, I will be able to create, access, manipulate, and use tuples effectively in Python programs.
