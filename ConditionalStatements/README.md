# 🔀 Conditional Statements

## 📖 About
This folder contains Python programs based on conditional statements. These programs demonstrate how to make decisions using different conditions.

## 📚 Topics Covered
- if Statement
- if-else Statement
- if-elif-else Statement
- Nested if Statements
- Logical Operators (and, or, not)
- Comparison Operators
- Conditional Expressions

## 🎯 Goal
To understand how decision-making works in Python and learn to write programs using different conditional statements.

## 📂 Programs
- if_statement.py
- if_else.py
- elif_statement.py
- nested_if.py
- logical_operators.py
- comparison_operators.py

## ✍️ Learning Outcome
After completing this folder, I will be able to use conditional statements to control the flow of Python programs and solve decision-making problems.
